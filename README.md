# Security Log Analysis Tool

Инструмент для автоматизированного анализа и корреляции логов безопасности с целью выявления подозрительной активности и типовых атак.

---

## Возможности

- Парсинг и анализ логов безопасности
- Корреляция событий по времени и источнику
- Обнаружение типовых атак (например, brute-force, sql-injection и т.д.)
- Генерация оповещений о подозрительной активности
- Сохранение событий и инцидентов в базу данных SQLite (имя по стандарту - logs.db)
- Консольный интерфейс управления
- Конфигурация правил обнаружения атак через JSON-файлы

---

## Архитектура проекта

Проект имеет модульную архитектуру и состоит из следующих основных компонентов:

- **main.py**  
  Точка входа в приложение, инициализация и запуск анализа

- **console_app.py**  
  Консольный интерфейс взаимодействия с пользователем

- **log_analyzer.py**  
  Основной модуль, где производится чтение логов, их парсинг, анализ и корреляция событий, обнаружение атак

- **database.py**  
  Работа с базой данных SQLite, хранение логов и инцидентов

Архитектура упрощает расширение функционала и добавление новых методов анализа.

---

## Алгоритм работы

1. Загрузка конфигурации
2. Чтение файла логов
3. Парсинг и нормализация записей
4. Корреляция событий по временным окнам и источникам
5. Обнаружение подозрительной активности
6. Сохранение результатов в базе данных
7. Вывод оповещений в консоль
